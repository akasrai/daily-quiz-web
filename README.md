### Installation

1. Clone project

```
git clone git@github.com:akasrai/daily-quiz-mobile.git
```

2. Install Dependencies

```
cd daily-quiz-web
yarn
```

3. Setup env

- Create new file `.env`
- Copy content from [.env.example](https://github.com/akasrai/daily-quiz-web/blob/master/.env)
- Add valid API url

4. Run development Server

```
npm start
```
